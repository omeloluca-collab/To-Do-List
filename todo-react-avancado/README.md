# To Do React Avançado

Aplicação de lista de tarefas (To Do List) desenvolvida com React utilizando recursos avançados:
Hooks, Hook customizado, Context API e Memoization.

## Tecnologias
- React
- Vite
- JavaScript
- CSS (Mobile First)

## Funcionalidades
- Adicionar tarefa
- Marcar como concluída
- Remover tarefa
- Filtrar: todas / pendentes / concluídas
- Persistência no localStorage

## Como rodar localmente
1. Clone o repositório
2. Instale as dependências:
   ```bash
   npm install
   npm run dev
